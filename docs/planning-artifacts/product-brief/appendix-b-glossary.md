# Appendix B: Glossary

| Term | Definition |
| ------ | ----------- |
| **BSUID** | Business-Scoped User ID — WhatsApp's new user identifier (replaces phone numbers for business messaging) |
| **NDPA** | Nigeria Data Protection Act 2023 — Primary data protection legislation |
| **GAID** | General Application and Implementation Directive — NDPC's enforcement instrument for NDPA |
| **DCMI** | Data Controller of Major Importance — Organizations processing significant personal data |
| **BSP** | Business Solution Provider — Intermediary for WhatsApp Business API access |
| **RLS** | Row-Level Security — PostgreSQL feature for multi-tenant data isolation |
| **DSAR** | Data Subject Access Request — User right to access/modify/delete their data |
| **PII** | Personally Identifiable Information |
| **Portfolio Pacing** | Meta's system for batching marketing campaigns based on feedback quality signals |
